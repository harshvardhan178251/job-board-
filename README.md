# TalentFlow

A modern MERN stack job board with:
- Smart filtered search
- AI match scoring
- One-click quick apply
- ATS pipeline tracking
- Automated screening questions
- Instant alerts
- Interview scheduling
- Real-time chat
- Resume parsing support
- Visual application dashboard

## Run locally

1. Install dependencies
   - npm install
   - npm --prefix frontend install
   - npm --prefix backend install
2. Start the app
   - npm run dev

## Deploy
- Connect the frontend to Vercel
- Add Vercel secrets to GitHub Actions
- Push to main to trigger CI/CD
